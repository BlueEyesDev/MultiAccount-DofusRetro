# MultiScreen DofusRetro
 Window management for multiple accounts

### Works for versions:
- **Dofus Retro v1.34.13**

### Shortcut Keyboard
- **F1** = Account 1
- **F2** = Account 2
- **F3** = Account 3
- **F4** = Account 4
- **F5** = Account 5
- **F6** = Account 6
- **F7** = Account 7
- **F8** = Account 8
