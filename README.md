# MultiScreen DofusRetro
 Window management for multiple accounts

### Works for versions:
- **Dofus Retro v1.35.***

### Shortcut Keyboard
- **F1** = Account 1
- **F2** = Account 2
- **F3** = Account 3
- **F4** = Account 4
- **F5** = Account 5
- **F6** = Account 6
- **F7** = Account 7
- **F8** = Account 8

### how to use the AutoConnect
- Create an Account.txt file at the root of the executable
- Then added a line for each account, example:
```
Account1	Password1
Account2	Password2
```
